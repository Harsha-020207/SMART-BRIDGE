About The Project

🏦 Project Overview

Banks receive thousands of credit card applications every day. This project automates the approval process using Artificial Intelligence and Machine Learning, reducing manual effort and improving decision accuracy.

🤖 Machine Learning

The system trains and compares multiple Machine Learning algorithms including Logistic Regression, Decision Tree, Random Forest and XGBoost. The best-performing model is automatically selected for real-time prediction.
